Way to propose branches for merging

How to find the proposer class?
 * is_compatible function for target branch
 * user can specify a proposer ("email-bundle", "launchpad", "github", ...) on
   the command-line or in config of the target branch

Options:
 * Source branch (and their public copy)
 * Target branch
 * Prerequisite branch

 * Proposed commit message
 * Merge proposal text
 * Suggested reviewers
 * Bugs fixed

Later, or never:
 * way to list proposals
 * way to approve proposals
 * way to comment on proposals
